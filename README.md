# ♟️ AI CHESS – Đấu trí cùng trí tuệ nhân tạo

Trang web AI Chess mô phỏng chơi cờ giữa:
- Người với Người
- AI với AI (2 bot tự đánh, người chỉ xem)

### Tính năng:
✅ Dark / Light mode  
✅ Bàn cờ thật có quân và ký hiệu hàng, cột  
✅ 2 chế độ hiển thị (Người / AI)  
✅ Người không thể can thiệp khi AI đấu  

### Cách mở rộng:
Để tạo AI thật:
- Sử dụng `fetch()` đến API ChatGPT, Grok, Gemini  
- Gửi FEN (trạng thái bàn cờ) và nhận nước đi hợp lệ  
- Thay phần random move bằng lời gọi API

### Chạy web:
1. Đưa toàn bộ file này lên GitHub Pages.  
2. URL: `https://<user>.github.io/ai-chess/`
